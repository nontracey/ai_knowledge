# 3.6 主流 Agent 开发框架概览

## 低代码/无代码框架

### Coze（扣子）

Coze 是字节跳动推出的零门槛 AI Bot 构建平台，通过拖拉拽操作设计 Agent 工作流程，添加功能插件，创建知识库，无需编程。

### Dify

Dify 是低代码平台，适合构建生产级 AI 应用，支持可视化编排和私有化部署，可设计复杂 RAG 流程，满足数据安全需求。

### n8n

n8n 是工作流自动化工具，可连接大量第三方服务，通过节点构建自动化工作流。

## 代码 Agent 框架

### LangChain

LangChain 是代码 Agent 框架，提供链、代理、内存、工具等组件，支持模块化开发，可灵活构建语言模型驱动应用。

### AutoGen

AutoGen 支持构建 Agent 团队，通过对话和协作解决复杂任务，如模拟软件开发团队，但上手门槛较高。

### Semantic Kernel

Semantic Kernel 是企业级框架，与 C#、Python 深度集成，提供安全、稳定的 AI 服务接口，适合嵌入企业应用。

### CrewAI

CrewAI 基于 LangChain 构建，强调角色和流程定义，适合模拟团队协作，流程确定性高，易于上手。

## 框架对比

| 框架 | 类型 | 特点 | 适用场景 |
|------|------|------|----------|
| Coze | 低代码 | 零门槛，插件丰富 | 快速创建对话 Bot |
| Dify | 低代码 | 开源，可视化编排 | 生产级 AI 应用 |
| LangChain | 代码框架 | 模块化，灵活性高 | 复杂 LLM 应用 |
| AutoGen | 代码框架 | 多 Agent 协作 | 复杂任务解决 |
| Semantic Kernel | 企业级框架 | C#/Python 集成 | 企业应用嵌入 |
| CrewAI | 代码框架 | 角色流程定义 | 团队协作模拟 |

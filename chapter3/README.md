# 第三章 AI Agent 应用开发基础

本章介绍 AI Agent 的基本概念、核心能力、发展历程，以及干流 Agent 开发框架的使用方法。

## 本章内容

- **3.1 什么是 AI Agent**：介绍 Agent 的定义、核心目标和人与环境的交互方式
- **3.2 Agent 的核心能力**：详解感知、思考、行动、记忆等核心能力
- **3.3 Agent 的发展历程**：梳理从符号 Agent 到基于大模型的 Agent 演进
- **3.4 Agent 的语言与多模态能力**：介绍语言交互和多模态融合能力
- **3.5 Agent 的四大特性与核心组件**：解析自主性、交互性、适应性、功能性四大特性
- **3.6 主流 Agent 开发框架概览**：对比低代码/无代码框架和代码 Agent 框架
- **3.7 Coze 零门槛构建智能体**：详解 Coze 平台的使用方法
- **3.8 Dify 开源 LLM 应用开发平台**：详解 Dify 的功能和使用方法

## 学习目标

- 理解 AI Agent 的基本概念和核心能力
- 了解 Agent 的发展历程和未来趋势
- 掌握 Agent 的语言交互和多模态能力
- 熟悉主流 Agent 开发框架的特点
- 掌握 Coze 和 Dify 的使用方法

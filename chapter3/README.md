# 第三章 AI Agent 基础

本章将介绍 AI Agent 的核心概念、能力和开发框架。AI Agent 是当前 AI 领域最热门的研究方向之一，它赋予了 AI 主动行动和工具使用的能力。

## 本章内容概览

| 小节 | 主题 | 核心知识点 |
|------|------|----------|
| 3.1 | 什么是 AI Agent | Agent 定义、感知-思考-行动-记忆循环 |
| 3.2 | Agent 的核心能力 | 感知、思考、行动、记忆四大能力详解 |
| 3.3 | Agent 的发展历程 | 从符号 Agent 到大模型 Agent |
| 3.4 | Agent 的语言与多模态能力 | 语言理解、生成、多模态融合 |
| 3.5 | Agent 的四大特性与组件 | 自主性、交互性、适应性、功能性与核心组件 |
| 3.6 | 主流 Agent 开发框架 | Coze、Dify、LangChain、AutoGen 等框架对比 |
| 3.7 | Coze 零门槛构建智能体 | 字节跳动 Coze 平台详解 |
| 3.8 | Dify 开源 LLM 应用平台 | Dify 平台功能与实战 |

## 学习目标

- 理解 AI Agent 的基本概念和核心能力
- 掌握 Agent 的感知-思考-行动工作循环
- 了解主流 Agent 开发框架的特点
- 能够使用 Coze 或 Dify 构建简单 Agent

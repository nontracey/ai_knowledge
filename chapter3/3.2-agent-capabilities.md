# 3.2 Agent 的核心能力

## 感知能力

### 信息转化

Agent 将外部信息转化为内部可理解的信号，从海量数据中识别规律和模式，准确辨认相关对象、事件和状态，为后续决策和行动提供基础。

### 感知重要性

感知能力是 Agent 自主行为的关键基础，没有有效感知，Agent 无法与环境互动，无法获取做出正确判断所需的信息。

### 数据支持

通过精准感知，Agent 为思考、决策和行动环节提供坚实且必要的数据支持和情境认知，如智能客服感知用户情绪，工业机器人感知零件位置。

## 思考能力

### 学习能力

Agent 利用机器学习技术，从与环境互动的经历中学习总结，积累新知识，优化决策和推理过程，提升适应性和性能。

### 推理能力

Agent 运用逻辑推理和模式识别，从已知信息推断未知信息，如通过用户行为推理潜在偏好，帮助其在不确定环境中提前应对。

### 决策能力

Agent 根据目标和任务，结合感知信息，运用内部机制形成明确行动决定，如在下棋时决策下一步棋的位置。

## 行动能力

### 执行操作

Agent 依据决策结果执行相应操作，如调用 API 查询天气、向其他 Agent 或用户提问、修改数据库数据等。

### 行动重要性

行动能力使 Agent 能够将决策转化为实际影响，控制和影响环境或系统，如机器人 Agent 的物理移动和抓取物体。

### 协同关系

行动能力与感知、思考能力协同：感知提供信息输入，思考做出决策，行动将决策转化为实际行为，共同构成 Agent 自主智能的核心闭环。

## Agent 记忆能力

将过去感知的信息、互动过程中积累的经验、学习到的各类知识稳固地存储起来。

### 信息存储

Agent 存储海量多类型信息，包括感知数据、决策内容、执行反馈等，通过合适技术手段和数据结构方便后续查阅和利用。

### 经验积累

Agent 在与环境互动中积累经验，学习新知识和技能，记录成功案例和失败教训，为未来决策提供参考。

### 知识推理

Agent 基于记忆进行知识推理，运用已学知识和经验分析新问题，如根据维修记录推理设备故障原因，提升应对复杂情况的能力。

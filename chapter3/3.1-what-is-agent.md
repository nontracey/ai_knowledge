# 3.1 什么是 AI Agent

## 人工智能愿景

人工智能旨在通过计算机程序或机器模仿、拓展和提升人类智能，涵盖解决复杂问题、学习新知识和适应新环境的能力，其研究范围广泛，从简单自动化任务到复杂决策分析。

## 传统 AI 的局限性

传统 AI 功能固化，只能在特定环境中完成预设任务，缺乏灵活性和自主适应环境变化的能力。这种局限性促使了 Agent 概念的诞生。

## Agent 核心目标

Agent 旨在推动人工智能从固定、被动状态向灵活、主动性演进，具备自主能力，能感知环境、思考决策并采取行动，如 ChatGPT 和 Cursor 中的 Agent 模式。

Agent 的四大核心能力：**感知、思考、行动、记忆**。

## Agent 工作模式

### 感知环境

自动驾驶 Agent 通过摄像头、雷达等传感器感知交通状况，包括车辆位置、行人动态、信号灯状态等，为决策提供实时输入。

### 做出决策

基于感知信息，Agent 进行复杂思考，如自动驾驶 Agent 根据交通流、车距、导航路线等因素决定加速、减速或转向。

### 采取行动

决策完成后，Agent 将决策转化为实际操作，如自动驾驶 Agent 控制油门、刹车和方向盘，确保行车安全和效率。

## 人与 Agent 的交互方式

Agent 与人类用户进行持续对话，接收人类指令并反馈执行结果，形成良性的交互循环。人类提供目标指导，Agent 自主规划执行路径。

# AI 从入门到应用开发实战

本书是一份面向零基础读者的 AI 学习指南，系统性地涵盖了从人工智能基础概念到大模型应用开发实战的完整知识体系。

## 本书特色

- **零基础友好**：从 AI 最基本的概念讲起，循序渐进，无需编程或数学预备知识
- **理论与实践结合**：每个概念都配有实际案例和代码示例
- **覆盖主流技术栈**：涵盖 LangChain、Semantic Kernel、AutoGen、Dify、Coze 等主流框架
- **项目驱动学习**：通过完整的实战项目巩固所学知识

## 适合读者

- 希望了解 AI 和大语言模型的初学者
- 想要掌握 AI 应用开发技能的开发者
- 对 RAG、Agent 等前沿技术感兴趣的技术人员
- 希望利用 AI 提升工作效率的各行业从业者

| 章节 | 主题 | 核心内容 |

| 第一章 | 人工智能通识基础 | AI 基本概念、发展历程与未来趋势 |

# 第一章 人工智能通识基础

本章介绍人工智能的基础概念、发展历程和未来趋势，帮助读者建立对 AI 技术的整体认知。

## 本章内容

- **1.1 什么是人工智能**：介绍 AI 的定义、实现方式、子领域以及图灵测试等评估方法
- **1.2 AI 的发展历程与趋势**：从符号主义到深度学习，梳理 AI 发展的四次浪潮，介绍机器学习、监督学习、无监督学习、强化学习等核心概念

## 学习目标

- 理解人工智能的基本定义和核心概念
- 了解 AI 发展的四个阶段：规则驱动、统计学习、深度学习、通用人工智能探索
- 掌握机器学习的基本分类：监督学习、无监督学习、强化学习
- 理解深度学习的基本原理和核心算法
- 了解人工智能的发展趋势和前沿方向

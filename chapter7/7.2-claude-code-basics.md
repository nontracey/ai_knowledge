# 7.2 Claude Code 基础入门

## Claude Code 概述

Claude Code 是 Anthropic 推出的命令行编程助手，可以帮助开发者完成代码编写、调试、重构等任务。

### 主要特点

- **智能代码生成**：根据自然语言描述生成代码
- **代码理解**：能够理解和分析现有代码
- **多语言支持**：支持 Python、JavaScript、TypeScript、Go、Rust 等多种语言
- **终端集成**：直接在终端中使用，无需切换应用

## 安装与配置

### 安装方式

Claude Code 可以通过 npm 或其他包管理器安装：

```bash
npm install -g @anthropic-ai/claude-code
```

### 认证配置

首次使用需要配置 API Key：

```bash
claude-code config set api-key YOUR_API_KEY
```

## 基本使用方法

### 启动 Claude Code

在终端中输入：

```bash
claude-code
```

### 常用命令

| 命令 | 说明 |
|------|------|
| `/help` | 显示帮助信息 |
| `/clear` | 清除对话历史 |
| `/model` | 切换 AI 模型 |
| `/skills` | 管理 Skills |
| `/exit` | 退出 Claude Code |

### 对话模式

Claude Code 支持两种对话模式：

1. **交互模式**：启动后直接输入问题
2. **单次模式**：通过命令行参数直接提问

```bash
# 交互模式
claude-code

# 单次模式
claude-code "解释这段代码的作用"
```

## 使用技巧

### 1. 明确描述需求

越清晰地描述需求，获得的帮助越准确。

### 2. 提供上下文

提供相关的代码片段或文件路径，帮助 Claude Code 更好地理解问题。

### 3. 分步骤提问

复杂问题可以拆分成多个小问题逐步解决。

### 4. 利用反馈优化

根据 Claude Code 的回答进行调整和追问。

## 代码操作能力

Claude Code 可以帮助你：

- **编写新代码**：根据描述生成代码
- **修复 bug**：分析错误并提供修复方案
- **代码重构**：优化代码结构和性能
- **添加注释**：为代码添加文档注释
- **编写测试**：生成单元测试和集成测试
- **代码审查**：分析代码质量并提供改进建议

## 配置选项

Claude Code 支持多种配置选项：

```json
{
  "model": "claude-3-5-sonnet-20241022",
  "temperature": 0.7,
  "max-tokens": 4096
}
```

可以通过 `claude-code config` 命令查看和修改配置。

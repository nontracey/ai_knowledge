# 第五章 LangChain 项目实战

本章通过一个完整的实战项目，展示如何使用 LangChain 开发 AI 应用，包括模型服务层、API 路由、业务层、角色会话管理、消息服务，以及 MCP 服务的集成。

## 本章内容

- **5.1 项目概述与架构设计**：介绍项目的整体架构和设计思路
- **5.2 模型服务层实现**：实现统一的模型提供商服务类
- **5.3 API 路由与业务层**：设计并实现 REST API 路由接口
- **5.4 角色与会话管理**：构建角色管理和会话管理服务
- **5.5 消息服务与对话 API**：构建消息服务，支持消息的发送、接收与历史记录管理
- **5.6 MCP 服务生态与交互流程**：介绍 MCP 协议和服务生态
- **5.7 MCP 文件系统服务实战**：详解 MCP 文件系统服务的使用
- **5.8 实战项目集成 MCP 设计思路**：介绍如何在项目中集成 MCP 服务

## 学习目标

- 掌握 LangChain 实战项目的开发流程
- 理解分层架构的设计思想
- 掌握 MCP 服务的集成方法
- 能够开发完整的 AI 应用后端服务

# 5.3 API 路由与业务层

## API 路由设计

### RESTful API 规范

| 方法 | 路径 | 说明 |
|------|------|------|
| POST | /api/chat | 发送对话请求 |
| POST | /api/chat/stream | 流式对话请求 |
| GET | /api/sessions | 获取会话列表 |
| POST | /api/sessions | 创建新会话 |
| DELETE | /api/sessions/{id} | 删除会话 |
| GET | /api/messages/{session_id} | 获取会话消息 |

## 路由实现

### 依赖注入

```python
from fastapi import Depends, HTTPException
from typing import Annotated

async def get_model_service():
    """获取模型服务"""
    return model_service

async def get_session_service():
    """获取会话服务"""
    return session_service

# 依赖类型别名
ModelService = Annotated[ModelService, Depends(get_model_service)]
SessionService = Annotated[SessionService, Depends(get_session_service)]
```

### 对话接口

```python
from fastapi import APIRouter, Depends
from pydantic import BaseModel
from typing import List, Optional

router = APIRouter(prefix="/api", tags=["chat"])

class ChatRequest(BaseModel):
    messages: List[dict]
    model_type: str = "deepseek"
    temperature: float = 0.7
    max_tokens: Optional[int] = None

class ChatResponse(BaseModel):
    session_id: str
    message: dict

@router.post("/chat", response_model=ChatResponse)
async def chat(
    request: ChatRequest,
    model_service: ModelService,
    session_service: SessionService
):
    """处理对话请求"""
    try:
        # 调用模型服务
        response = await model_service.chat(
            model_type=request.model_type,
            messages=request.messages,
            temperature=request.temperature,
            max_tokens=request.max_tokens
        )
        
        # 创建新消息
        message = {
            "role": "assistant",
            "content": response
        }
        
        # 保存消息到会话
        session_id = request.messages[0].get("session_id", "default")
        await session_service.add_message(session_id, message)
        
        return ChatResponse(
            session_id=session_id,
            message=message
        )
    except Exception as e:
        raise HTTPException(status_code=500, detail=str(e))
```

### 流式对话接口

@router.post("/chat/stream")
async def chat_stream(

### 会话管理接口

```python
@router.get("/sessions")
async def list_sessions(session_service: SessionService):
    """获取会话列表"""
    sessions = await session_service.list_sessions()
    return {"sessions": sessions}

@router.post("/sessions")
async def create_session(session_service: SessionService):
    """创建新会话"""
    session = await session_service.create_session()
    return session

@router.delete("/sessions/{session_id}")
async def delete_session(
    session_id: str,
    session_service: SessionService
):
    """删除会话"""
    await session_service.delete_session(session_id)
    return {"status": "ok"}
```

## 错误处理

```python
from fastapi.responses import JSONResponse
from fastapi.exceptions import RequestValidationError

@router.exception_handler(RequestValidationError)
async def validation_exception_handler(request, exc):
    return JSONResponse(
        status_code=422,
        content={"detail": exc.errors()}
    )

@router.exception_handler(Exception)
async def general_exception_handler(request, exc):
    return JSONResponse(
        status_code=500,
        content={"detail": "Internal server error"}
    )
```

# 5.5 检索策略与完整 RAG 链

## 检索器接口

LangChain 的检索器提供统一接口：

```python
retriever = vectorstore.as_retriever(
    search_kwargs={"k": 4}  # 返回 4 个结果
)

results = retriever.invoke("用户问题")
```

## 检索类型

### 1. 相似性搜索

最基本的检索方式：

```python
results = vectorstore.similarity_search(
    query="查询内容",
    k=4  # 返回 4 个最相似结果
)
```

**特点：**

- 基于向量相似度
- 适合通用问答
- 简单高效

### 2. 最大边际相关性（MMR）

在相关性和多样性间平衡：

```python
results = vectorstore.max_marginal_relevance_search(
    query="查询内容",
    k=4,           # 最终返回 4 个
    fetch_k=10,    # 候选 10 个
    lambda_mult=0.5  # 多样性权重
)
```

**解决问题：**

避免返回内容高度重复的文档，提高结果多样性。

### 3. 相似度阈值

只返回高相关性结果：

```python
results = vectorstore.similarity_search_with_score(
    query="查询内容",
    k=4
)

# 过滤低相关性结果
filtered = [doc for doc, score in results if score < 0.7]
```

**适用场景：**

- 需要高质量结果
- 过滤不相关回答

## 完整 RAG 链

### 基础 RAG 链

```python
from langchain_core.prompts import ChatPromptTemplate
from langchain_core.output_parsers import StrOutputParser

# 提示模板
prompt = ChatPromptTemplate.from_messages([
    ("system", "基于以下上下文回答问题：\n{context}"),
    ("human", "{question}")
])

# 输出解析器
output_parser = StrOutputParser()

# 构建链
rag_chain = (
    {"context": retriever, "question": RunnablePassthrough()}
    | prompt
    | model
    | output_parser
)

# 执行
result = rag_chain.invoke("什么是人工智能？")
```

### 带历史的多轮对话链

```python
from langchain_core.messages import HumanMessage, AIMessage

# 对话历史
chat_history = [
    HumanMessage(content="人工智能是什么？"),
    AIMessage(content="人工智能是...")
]

# 构建提示
prompt = ChatPromptTemplate.from_messages([
    ("system", "基于上下文和对话历史回答：\n{context}"),
    MessagesPlaceholder("chat_history"),
    ("human", "{question}")
])

# 带历史的链
rag_chain = (
    {
        "context": retriever,
        "chat_history": lambda x: chat_history,
        "question": RunnablePassthrough()
    }
    | prompt
    | model
    | output_parser
)
```

## 检索策略优化

### 混合检索

结合多种检索方式：

```python
# 关键词检索 + 向量检索
from langchain.retrievers import EnsembleRetriever

retriever = EnsembleRetriever(
    retrievers=[keyword_retriever, vector_retriever],
    weights=[0.3, 0.7]
)
```

### 查询改写

改写用户问题提升检索效果：

```python
rewrite_prompt = ChatPromptTemplate.from_messages([
    ("system", "将用户问题改写为独立、清晰的查询。"),
    ("human", "{question}")
])

rewrite_chain = rewrite_prompt | model | output_parser
rewritten_query = rewrite_chain.invoke({"question": user_question})
```

### 重排（Rerank）

对检索结果二次排序：

```python
from langchain.retrievers.document_compressors import CrossEncoderReranker

reranker = CrossEncoderReranker(
    model="BAAI/bge-reranker-base",
    top_n=3
)

compressor = ContextualCompressionRetriever(
    base_compressor=reranker,
    base_retriever=vector_retriever
)
```

## 高级检索策略

### 两阶段检索

| 阶段 | 目的 | 方法 |
|------|------|------|
| 召回 | 扩大候选范围 | 多查询、混合检索 |
| 精排 | 提高精度 | 重排、过滤 |

### 自查询检索

根据元数据过滤：

```python
from langchain.retrievers import SelfQueryRetriever

retriever = SelfQueryRetriever.from_llm(
    llm=model,
    vectorstore=vectorstore,
    document_contents="技术文档",
    metadata_field_info=metadata_fields
)
```

## 小结

检索是 RAG 系统的核心环节。相似性搜索、MMR 和相似度阈值各有适用场景。完整 RAG 链通过 LCEL 将检索、提示和生成串联起来。高级检索策略如混合检索、查询改写和重排可以显著提升系统效果。

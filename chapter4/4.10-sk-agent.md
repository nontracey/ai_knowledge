# 4.10 Semantic Kernel Agent 框架

## 框架核心架构

- **内核（Kernel）**：功能强大的依赖注入容器，统一管理运行 AI 应用所需的所有服务和插件
- **智能体（Agent）**：旨在自主或半自主地执行任务的软件实体，接收输入、处理信息、采取行动达成目标
- **智能体线程（AgentThread）**：一个智能体可同时参与多个独立对话，负责维护对话状态和上下文

## 创建第一个智能体应用

### 环境准备

创建 .NET 控制台应用程序，配置 API Key。

### 创建内核与智能体

```csharp
const string instructions = "你是一个乐于助人的 AI 助手，名叫"朝小夕"。";

var agent = new ChatCompletionAgent()
{
    Kernel = kernel,
    Name = "朝小夕",
    Instructions = instructions,
    Arguments = new KernelArguments(new OpenAIPromptExecutionSettings
    {
        FunctionChoiceBehavior = FunctionChoiceBehavior.Auto()
    })
};
```

### 管理对话状态

```csharp
var agentThread = new ChatHistoryAgentThread();
```

### 调用智能体

```csharp
var userMessage = new ChatMessageContent(AuthorRole.User, userInput);

await foreach (var response in agent.InvokeStreamingAsync(userMessage, agentThread))
{
    Console.Write(response.Message.Content);
}
```

## 智能体自动调用插件

### 创建插件

```csharp
public class WeatherPlugin
{
    [KernelFunction]
    [Description("获取指定城市当前的天气信息")]
    public string GetWeather(
        [Description("要查询的城市名称")] string city)
    {
        return city switch
        {
            "北京" => "晴，25°C",
            "上海" => "多云，28°C",
            _ => $"抱歉，我不知道城市"{city}"的天气。"
        };
    }
}

// 注册插件
builder.Plugins.AddFromType<WeatherPlugin>();
```

## 通过模板构建 Agent

### 指令模板化

```csharp
var agent = new ChatCompletionAgent()
{
    Instructions = "请写一个关于{{$topic}}的故事，故事长度为{{$length}}句话。",
    Arguments = new KernelArguments()
    {
        { "topic", "狗" },
        { "length", "3" }
    }
};
```

### YAML 声明式定义

```yaml
name: GenerateStory
description: 一个可以根据指定的主题和句子数量来生成故事的 Agent
template_format: semantic-kernel
template: |
  你的任务是扮演一个富有创造力的故事大王。
  请根据下面的要求创作一个故事：
  - 故事主题：{{$topic}}
  - 故事长度：恰好 {{$length}} 句话。
input_variables:
  - name: topic
    description: 故事的主题
    is_required: true
  - name: length
    description: 故事需要包含的句子数量
    is_required: true
    default: "3"
```

### 加载 YAML 定义

```csharp
var yamlContent = await File.ReadAllTextAsync("./GenerateStory.yaml");
var templateConfig = KernelFunctionYaml.ToPromptTemplateConfig(yamlContent);

var agent = new ChatCompletionAgent(templateConfig, new KernelPromptTemplateFactory())
{
    Kernel = kernel,
    Arguments = new KernelArguments() { { "topic", "一只迷路的小狗" } }
};
```

## 最佳实践建议

1. **统一文件组织**：创建专门目录存放 YAML 文件
2. **清晰命名**：制定一致且具有描述性的命名规范
3. **善用描述字段**：在 YAML 的 description 字段中编写清晰准确的描述

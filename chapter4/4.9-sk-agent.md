# 4.9 Semantic Kernel Agent 框架

## Agent 概述

### Agent 与 Kernel 的关系

- **Kernel**：核心编排器，管理服务和插件
- **Agent**：智能体，负责执行具体任务

### Agent 的特点

- 自主执行任务
- 维护对话状态
- 调用工具完成任务

## 创建 Agent

### 基本步骤

```csharp
// 1. 创建内核
var kernel = Kernel.CreateBuilder()
    .AddOpenAIChatCompletion("gpt-4", apiKey)
    .Build();

// 2. 定义指令
var instructions = @"你是一个乐于助人的 AI 助手，名叫'小智'。
尽你所能回答用户的所有问题。";

// 3. 创建 Agent
var agent = new ChatCompletionAgent
{
    Kernel = kernel,
    Name = "小智",
    Instructions = instructions,
    Arguments = new KernelArguments(new OpenAIPromptExecutionSettings
    {
        FunctionChoiceBehavior = FunctionChoiceBehavior.Auto()
    })
};
```

### 调用 Agent

```csharp
// 创建对话线程
var agentThread = new ChatHistoryAgentThread();

// 首次交互
var userMessage = new ChatMessageContent(AuthorRole.User, "你好！");
await foreach (var response in agent.InvokeStreamingAsync(userMessage, agentThread))
{
    Console.Write(response.Message.Content);
}

// 继续对话
var followUp = new ChatMessageContent(AuthorRole.User, "你能做什么？");
await foreach (var response in agent.InvokeStreamingAsync(followUp, agentThread))
{
    Console.Write(response.Message.Content);
}
```

## 自动调用工具

### 注册工具

```csharp
public class WeatherPlugin
{
    [KernelFunction]
    [Description("获取指定城市的天气信息")]
    public string GetWeather(
        [Description("要查询的城市名称")] string city)
    {
        return city switch
        {
            "北京" => "晴，25°C",
            "上海" => "多云，28°C",
            _ => $"抱歉，不知道{city}的天气"
        };
    }
}

// 注册
kernel.Plugins.AddFromType<WeatherPlugin>("Weather");
```

### 配置自动调用

```csharp
var agent = new ChatCompletionAgent
{
    Kernel = kernel,
    Name = "天气助手",
    Instructions = "你是一个天气助手，可以查询城市天气。",
    Arguments = new KernelArguments(new OpenAIPromptExecutionSettings
    {
        FunctionChoiceBehavior = FunctionChoiceBehavior.Auto()
    })
};

// 测试
var result = await agent.InvokeAsync(
    new ChatMessageContent(AuthorRole.User, "北京今天天气怎么样？"),
    agentThread
);
```

## 模板化 Agent

### 参数化指令

```csharp
var agent = new ChatCompletionAgent
{
    Instructions = "请为{{$audience}}写一篇关于{{$topic}}的简介。",
    Arguments = new KernelArguments
    {
        ["audience"] = "高中生",
        ["topic"] = "人工智能"
    }
};
```

### YAML 定义

创建 `GenerateStory.yaml`：

```yaml
name: GenerateStory
description: 生成故事
template_format: semantic-kernel
template: |
  你的任务是扮演一个故事大王。
  请根据以下要求创作故事：
  - 主题：{{$topic}}
  - 长度：{{$length}}句话
template:
  input_variables:
    - name: topic
      description: 故事主题
      is_required: true
    - name: length
      description: 句子数量
      default: "3"
```

### 加载 YAML Agent

```csharp
var yamlContent = await File.ReadAllTextAsync("./GenerateStory.yaml");
var templateConfig = KernelFunctionYaml.ToPromptTemplateConfig(yamlContent);

var agent = new ChatCompletionAgent(templateConfig, new KernelPromptTemplateFactory())
{
    Kernel = kernel,
    Arguments = new KernelArguments { ["topic"] = "太空探险" }
};
```

## 最佳实践

### 1. 清晰的指令

```csharp
// 详细指令
var instructions = @"你是一个专业的数学辅导老师。
- 解释问题时使用简单的语言
- 提供逐步的推导过程
- 在适当的地方给出例子";
```

### 2. 适当的参数设置

```csharp
var settings = new OpenAIPromptExecutionSettings
{
    Temperature = 0.7,      // 创造性
    MaxTokens = 1000,      // 长度限制
    FunctionChoiceBehavior = FunctionChoiceBehavior.Auto()
};
```

### 3. 错误处理

```csharp
try
{
    var result = await agent.InvokeAsync(message, agentThread);
}
catch (KernelException ex)
{
    Console.WriteLine($"Agent 执行错误: {ex.Message}");
}
```

## 小结

Semantic Kernel Agent 提供了创建智能对话 Agent 的完整框架。通过定义指令、配置工具调用、使用模板化指令，可以构建功能强大的 AI Agent。

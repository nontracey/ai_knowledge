# 4.4 提示模板

## 提示模板的概念

提示模板是包含固定文本和动态占位符的字符串，用于生成标准化的提示。

### 为什么需要提示模板

- **一致性**：确保相同类型的任务使用相同的提示结构
- **复用性**：一次定义，多次使用
- **可维护性**：集中管理提示词，易于修改

## 字符串提示模板

### 基本用法

```python
from langchain.prompts import PromptTemplate

template = PromptTemplate.from_template(
    "请将以下英文翻译成中文：{text}"
)

prompt = template.invoke({"text": "Hello, world!"})
```

### 自定义模板

```python
template = PromptTemplate(
    input_variables=["topic", "length"],
    template="请写一篇关于{topic}的文章，长度约{length}字。"
)

prompt = template.invoke({"topic": "人工智能", "length": "500"})
```

## 聊天提示模板

### 多角色对话

```python
from langchain.prompts import ChatPromptTemplate

template = ChatPromptTemplate.from_messages([
    ("system", "你是一个{role}助手。"),
    ("user", "请问{question}？")
])

prompt = template.invoke({
    "role": "技术",
    "question": "什么是机器学习"
})
```

### MessagesPlaceholder

插入对话历史：

```python
from langchain.prompts import MessagesPlaceholder

template = ChatPromptTemplate.from_messages([
    ("system", "你是一个有帮助的助手。"),
    MessagesPlaceholder(variable_name="chat_history"),
    ("human", "{question}")
])

prompt = template.invoke({
    "chat_history": [
        ("user", "人工智能是什么？"),
        ("assistant", "人工智能是...")
    ],
    "question": "它有哪些应用？"
})
```

## 少样本提示模板

### FewShotPromptTemplate

```python
from langchain.prompts import FewShotPromptTemplate, PromptTemplate

examples = [
    {"input": "iPhone 14", "output": "突破性的创新设计，卓越的性能表现。"},
    {"input": "MacBook Air", "output": "轻薄优雅的便携之选，出众的续航能力。"}
]

example_prompt = PromptTemplate.from_template(
    "输入：{input}\n输出：{output}"
)

few_shot_prompt = FewShotPromptTemplate(
    examples=examples,
    example_prompt=example_prompt,
    prefix="请按照以下格式撰写产品描述：",
    suffix="\n\n输入：{input}\n输出：",
    input_variables=["input"]
)
```

## 管道提示模板

将复杂提示分解为多个子模板：

```python
from langchain.prompts import PipelinePromptTemplate

# 子模板
introduction = PromptTemplate.from_template("你是一个{role}专家。")
reflection = PromptTemplate.from_template(
    "你已经介绍了{topic}的基础知识。{reflection_prompt}"
)
conclusion = PromptTemplate.from_template(
    "总结以上内容：{topic}的核心要点是{key_points}。"
)

# 管道组合
pipeline_prompt = PipelinePromptTemplate(
    pipeline_prompts=[
        ("introduction", introduction),
        ("reflection", reflection),
        ("conclusion", conclusion)
    ],
    final_prompt PromptTemplate.from_template(
        "{introduction}\n\n{reflection}\n\n{conclusion}"
    ),
    input_variables=["role", "topic", "reflection_prompt", "key_points"]
)
```

## 预填充变量

### 静态预填充

```python
from langchain.prompts import PromptTemplate

base_template = PromptTemplate.from_template(
    "今天是{date}，请生成每日报表。"
)

# 预填充固定值
filled_template = base_template.partial(date="2024-01-15")
prompt = filled_template.invoke({})
```

### 函数动态填充

```python
from datetime import datetime

def get_date():
    return datetime.now().strftime("%Y-%m-%d")

template = PromptTemplate(
    input_variables=["date"],
    template="今天是{date}，请生成每日报表。",
    partial_variables={"date": get_date}  # 自动填充
)

prompt = template.invoke({})  # date 自动获取
```

## 模板持久化

### 保存为 JSON

```python
template = ChatPromptTemplate.from_messages([
    ("system", "你是一个{role}助手。"),
    ("human", "{question}")
])

# 保存
template.save("prompt_template.json")

# 加载
from langchain.prompts import load_prompt
loaded = load_prompt("prompt_template.json")
```

### LangChain Prompt Hub

```python
from langchain import hub

# 拉取模板
prompt = hub.pull("rlatensch/analyze-code")
```

## 小结

提示模板是组织和管理提示词的有效方式。从简单的字符串模板到复杂的多角色对话模板、管道模板，LangChain 提供了丰富的支持。掌握这些技巧可以大大提高提示词工程的可维护性和复用性。

# 4.4 提示模板

## 提示模板概念

提示模板是包含固定文本和动态占位符的字符串，用于生成标准化提示。

### 优势

- 提高一致性、效率
- 改善模型性能
- 增强灵活性与可扩展性

## 字符串提示模板

```python
from langchain_core.prompts import PromptTemplate

prompt = PromptTemplate.from_template("给我讲一个关于{topic}的笑话")
print(prompt.invoke({"topic": "程序员"}))
```

## 聊天提示模板

```python
from langchain_core.prompts import ChatPromptTemplate, MessagesPlaceholder

prompt = ChatPromptTemplate.from_messages([
    ("system", "你是一个乐于助人的{assistant_type}。"),
    MessagesPlaceholder("chat_history"),
    ("human", "{question}")
])
```

## 少样本提示模板

```python
from langchain_core.prompts import FewShotChatMessagePromptTemplate

examples = [
    {"input": "iPhone", "output": "突破性的创新设计，卓越的性能表现。"},
    {"input": "MacBook", "output": "轻薄优雅的便携之选，出众的续航能力。"}
]

few_shot_prompt = FewShotChatMessagePromptTemplate(
    examples=examples,
    example_prompt=example_prompt
)
```

## 管道提示模板

允许将复杂提示分解为多个逻辑独立且可复用的子模板，通过流水线方式串联起来。

```python
from langchain_core.prompts import PipelinePromptTemplate

full_prompt = PipelinePromptTemplate(
    pipeline_prompts=[
        ("role", role_prompt),
        ("context", context_prompt),
        ("question", question_prompt)
    ],
    final_prompt=final_prompt
)
```

## 预填充变量

### 使用静态值预填充

```python
prompt = prompt.partial(name="张三")
```

### 使用函数动态填充

```python
from datetime import datetime

prompt = prompt.partial(
    date=lambda: datetime.now().strftime("%Y-%m-%d")
)
```

## 模板的持久化

LangChain 支持将提示模板持久化到本地文件系统的功能，支持 JSON 和 YAML 两种格式。

```python
from langchain_core.prompts import load_prompt

# 保存
prompt.save("prompt.json")

# 加载
prompt = load_prompt("prompt.json")
```

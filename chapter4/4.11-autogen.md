# 4.11 AutoGen 多智能体框架

## AutoGen 概述

AutoGen 是微软推出的开源框架，专注于构建基于大语言模型的多智能体应用。

### 核心特点

- **多智能体协作**：多个 Agent 协同工作
- **对话驱动**：通过对话协作解决问题
- **灵活扩展**：支持自定义 Agent 角色

### AutoGen vs LangChain

| 维度 | AutoGen | LangChain |
|------|---------|-----------|
| 核心抽象 | 对话 | 链 |
| 多 Agent | 原生支持 | 通过 Agent 实现 |
| 学习曲线 | 中等 | 中等 |

## 架构分层

| 层级 | 说明 |
|------|------|
| Studio | 无代码 UI（AutoGen Studio） |
| AgentChat | 高级 API |
| Core | 底层事件驱动 API |
| Extensions | 插件扩展 |

## 核心概念

### 可对话智能体

能够发送和接收消息的实体：

```python
from autogen_agentchat.agents import AssistantAgent

agent = AssistantAgent(
    name="assistant",
    model_client=model_client,
    tools=[search_tool, calculator_tool]
)
```

### 用户代理

代表人类用户的 Agent：

```python
from autogen_agentchat.agents import UserProxyAgent

user_proxy = UserProxyAgent(
    name="user_proxy",
    code_execution_config={"executor": local_executor}
)
```

## 对话模式

### 双智能体对话

```python
import asyncio
from autogen_agentchat.agents import AssistantAgent
from autogen_agentchat.ui import Console

# 创建 Agent
assistant = AssistantAgent(
    name="assistant",
    model_client=model_client,
    tools=[weather_tool]
)

user_proxy = UserProxyAgent(name="user")

# 运行对话
async def main():
    result = await assistant.run_chat(
        task="北京今天的天气怎么样？适合出门吗？",
        user_proxy=user_proxy
    )
    print(result)

asyncio.run(main())
```

### 顺序对话

多个 Agent 顺序执行：

```python
# 链式对话
result1 = await agent1.run_chat(task="任务1")
result2 = await agent2.run_chat(task=f"基于{result1}执行任务2")
```

### 群聊

多个 Agent 在共享空间中交流：

```python
from autogen_agentchat.teams import GroupChat

team = GroupChat(
    participants=[agent1, agent2, agent3],
    max_turns=10
)

await team.run(task="完成一个复杂项目")
```

## 团队协作

### 选择式小组对话

LLM 动态选择下一个发言者：

```python
from autogen_agentchat.teams import SelectorGroupChat

team = SelectorGroupChat(
    participants=[writer, reviewer, editor],
    selector_func=selector_function  # 自定义选择逻辑
)
```

### 轮询式小组对话

固定顺序轮流发言：

```python
from autogen_agentchat.teams import RoundRobinGroupChat

team = RoundRobinGroupChat(
    participants=[writer, reviewer]
)
```

## Swarm 集群模式

### 去中心化任务传递

Agent 直接移交任务给下一个：

```python
from autogen_agentchat.agents import handoff

# Agent A 移交任务给 Agent B
await agent_a.send(
    handoff(message="任务移交", target=agent_b)
)
```

### 示例：旅行客服

```
用户咨询 → 旅行客服 → 判断类型
                        ├── 退款请求 → 退款处理员
                        └── 预订咨询 → 预订专员
```

## Magentic-One

### 概述

Magentic-One 是基于 AutoGen 的预构建多智能体系统。

### 组件

| Agent | 功能 |
|-------|------|
| Orchestrator | 任务规划和协调 |
| WebSurfer | 网页浏览 |
| FileSurfer | 文件管理 |
| Coder | 代码编写 |
| ComputerTerminal | 代码执行 |

### 使用示例

```python
from autogen_ext.agents.magentic_one import MagenticOne

async def main():
    async with MagenticOne(log_level=logging.INFO) as agent:
        result = await agent.run(
            task="帮我分析这个 CSV 文件并生成报告"
        )
        print(result)

asyncio.run(main())
```

## GraphFlow 工作流

### 确定性工作流引擎

```python
from autogen_agentchat.graph import DiGraphBuilder, END

builder = DiGraphBuilder()

# 添加节点
builder.add_node(agent1, name="step1")
builder.add_node(agent2, name="step2")

# 添加边
builder.add_edge("step1", "step2")
builder.add_edge("step2", END)

graph = builder.build()
```

### 条件边

```python
builder.add_conditional_edge(
    "evaluate",
    lambda state: "approve" if state["score"] > 0.8 else "revise"
)
```

### 并行与合流

```python
# 并行处理
builder.add_edge("analyze", ["content_review", "style_review"])

# 合流
builder.add_edge(["content_review", "style_review"], "finalize")
```

## 小结

AutoGen 提供了强大的多智能体协作能力。从双 Agent 对话到群聊，从 Swarm 到 Magentic-One，可以构建各种复杂度的多 Agent 应用。GraphFlow 则提供了确定性工作流的能力，与动态 Agent 协作形成互补。

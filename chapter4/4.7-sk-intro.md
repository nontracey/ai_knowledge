# 4.7 Semantic Kernel 概述与环境搭建

## 什么是 Semantic Kernel

Semantic Kernel 是微软推出的轻量级 SDK，用于将大语言模型集成到应用程序中。

### 与 LangChain 的对比

| 维度 | Semantic Kernel | LangChain |
|------|-----------------|-----------|
| 设计理念 | 企业级、稳定性 | 社区驱动、灵活性 |
| 语言 | C# / Python | Python / JavaScript |
| 适用场景 | .NET 企业应用 | 快速原型、研究 |

### 核心优势

- **企业级**：稳定、安全、可审计
- **与 .NET 深度集成**：无缝集成到现有 .NET 应用
- **轻量级**：易于学习和使用
- **可扩展**：插件系统支持灵活扩展

## 核心架构

### 三大核心组件

```
┌─────────────────────────────────────┐
│              Kernel                  │  核心编排器
├─────────────────────────────────────┤
│  Plugins │ Memory │ Services          │  功能扩展
└─────────────────────────────────────┘
```

### Kernel（内核）

- 接收请求，编排插件
- 管理 AI 服务
- 执行任务

### Plugins（插件）

- **语义函数**：由提示词驱动的函数
- **原生函数**：C# 代码实现的函数

### Memory（记忆）

- 维护对话上下文
- 支持知识检索

## 环境准备

### .NET 项目创建

```bash
# 创建控制台项目
dotnet new console -n MyAIApp
cd MyAIApp

# 添加 NuGet 包
dotnet add package Microsoft.SemanticKernel
```

### 配置 API 密钥

```bash
# 初始化用户机密
dotnet user-secrets init

# 设置 API 密钥
dotnet user-secrets set "OpenAI:ApiKey" "your-api-key"
dotnet user-secrets set "OpenAI:ModelId" "gpt-4"
```

## 快速开始

### 基本用法

```csharp
using Microsoft.SemanticKernel;

// 创建内核
var builder = Kernel.CreateBuilder();
builder.AddOpenAIChatCompletion(
    modelId: "gpt-4",
    apiKey: "your-api-key"
);

var kernel = builder.Build();

// 调用
var result = await kernel.InvokePromptAsync("你好，请介绍一下你自己。");

Console.WriteLine(result);
```

### 完整示例

```csharp
using Microsoft.SemanticKernel;
using Microsoft.SemanticKernel.Connectors.OpenAI;

class Program
{
    static async Task Main(string[] args)
    {
        // 1. 创建内核
        var builder = Kernel.CreateBuilder();
        builder.AddOpenAIChatCompletion(
            modelId: "gpt-4",
            apiKey: "your-api-key"
        );
        var kernel = builder.Build();

        // 2. 定义提示词
        var prompt = @"请将以下英文翻译成中文：{{$input}}";

        // 3. 创建函数
        var translateFunction = kernel.CreateFunctionFromPrompt(
            promptTemplate: prompt,
            functionName: "Translate",
            description: "翻译英文到中文"
        );

        // 4. 执行
        var result = await kernel.InvokeAsync(
            translateFunction,
            new KernelArguments { ["input"] = "Hello, world!" }
        );

        Console.WriteLine(result);
    }
}
```

## 核心概念

### KernelArguments

传递参数的容器：

```csharp
var arguments = new KernelArguments
{
    ["input"] = "用户输入",
    ["temperature"] = 0.7,
    ["max_tokens"] = 1000
};
```

### PromptExecutionSettings

配置模型参数：

```csharp
var settings = new OpenAIPromptExecutionSettings
{
    Temperature = 0.7,
    TopP = 0.9,
    MaxTokens = 1000
};
```

## 小结

Semantic Kernel 提供了微软企业级的 AI 集成方案。通过 Kernel、Plugins 和 Memory 三大核心组件，可以构建强大的 AI 应用。其与 .NET 的深度集成，特别适合在企业级 .NET 应用中集成 AI 能力。

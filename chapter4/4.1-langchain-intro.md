# 4.1 LangChain 概述与环境准备

## LangChain 是什么

LangChain 是一个基于大语言模型的应用开发框架，提供工具、组件和接口，简化 AI 应用的开发过程。

### 语言版本

官方提供 Python 和 JavaScript 版本，Python 版本更受重视，社区还开发了 Go、Java 等其他语言版本。

### 大语言模型两大能力

1. **推理能力**：基于大模型的知识储备进行推理和决策
2. **工具调用能力**：能够调用外部工具和 API

## 为什么使用 LangChain

### 核心价值

- **数据感知**：能与多种数据源连接，将外部数据作为知识库
- **代理交互**：应用可主动与环境互动，如自动规划行程、总结邮件内容
- **解决痛点**：针对大模型内部数据过时、无法联网、Token 限制、无法调用 API 等问题提供解决方案

### 应用场景

- **智能客服**：构建特定领域智能问答机器人
- **文档摘要应用**：快速生成高质量摘要和目录
- **编程助手**：自动生成代码片段，分析代码质量
- **学习助手**：提供学业支持，口语陪练
- **智能私人助理**：完成繁琐任务，如旅行计划、预订机票酒店

## LangChain 架构

LangChain 共分三大类：
- 大语言模型
- 聊天模型
- 嵌入式模型

## 开发前的准备

### Anaconda 介绍

Anaconda 是数据科学、机器学习的开源 Python 发行版，简化环境安装和管理，包含包管理器、虚拟环境、科学计算库、机器学习库、集成开发环境等。

### 安装和使用

1. 推荐使用清华大学镜像源下载 Anaconda
2. 在 Anaconda Navigator 中创建专属环境
3. 安装 PyCharm 作为 IDE，配置为使用 Anaconda 环境

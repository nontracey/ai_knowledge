# 4.5 链（Chain）

## 链的定义

LangChain 中最核心的概念，指将多个模块化组件按特定顺序链接起来执行多步骤任务的序列。

### 工作流程

接受用户的输入 → 使用提示模板格式化输入 → 将提示发送模型 → 输出解析器

### 核心组件

- **提示模板**：负责构建和格式化模型的输入
- **模型**：链的核心引擎，包括大型语言模型和聊天模型
- **输出解析器**：将模型返回的原始输出转换为更结构化的格式

## LangChain 表达式语言（LCEL）

采用声明式的管道操作符 `|` 连接组件，具有以下优势：

- **可组合性**：易于组合复杂流程
- **可读性**：代码清晰易懂
- **灵活性**：支持灵活配置
- **原生支持高级功能**：内置流式处理、异步执行等
- **可观察性**：支持 LangSmith 调试

## 基础链

最基础的链结构：`prompt | model | output_parser`

```python
from langchain_core.output_parsers import StrOutputParser
from langchain_core.prompts import ChatPromptTemplate

chain = prompt | model | StrOutputParser()
result = chain.invoke({"topic": "人工智能"})
```

## 顺序链

将多个独立链串联形成复杂处理流水线，前一链输出是后一链输入。

### 简单顺序链

```python
from langchain_core.runnables import RunnableLambda

chain = (
    first_chain
    | RunnableLambda(lambda x: {"story": x})
    | second_chain
)
```

### 通用顺序链

处理多个输入输出变量，保留中间结果。

```python
from langchain.chains import LLMChain
from langchain_core.runnables import RunnablePassthrough

chain = (
    {"topic": RunnablePassthrough()}
    | first_chain
    | {"input": lambda x: x["story"], "language": lambda x: x["language"]}
    | second_chain
)
```

## 路由链

分析输入后将任务路由到多个预定义目标链中的某一个去执行。

```python
from langchain_core.runnables import RunnableLambda

def route_func(x):
    if "数学" in x["topic"]:
        return "math_chain"
    return "general_chain"

chain = (
    {"topic": lambda x: x["topic"]}
    | RunnableLambda(route_func)
)
```

## API 链

使 LLM 与外部 API 交互，获取实时信息或执行操作。

```python
from langchain.chains import APIChain

api_chain = APIChain.from_llm_and_api_url(llm, api_url, headers={"Authorization": f"Bearer {api_key}"})
```

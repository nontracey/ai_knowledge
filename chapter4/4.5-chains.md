# 4.5 链（Chain）

## 链的定义

链是 LangChain 的核心概念，将多个模块组件按特定顺序链接起来处理复杂任务。

### 简单理解

```
输入 → 提示模板 → 模型 → 输出解析器 → 输出
```

### LCEL 表达式

LangChain 使用管道操作符 `|` 连接组件：

```python
chain = prompt | model | output_parser
result = chain.invoke({"input": "..."})
```

## 链的类型

### 基础链

#### LLMChain

最基础的链：

```python
from langchain.chains import LLMChain

chain = LLMChain(
    llm=model,
    prompt=prompt
)
result = chain.run("用户问题")
```

#### 翻译链示例

```python
from langchain.chains import LLMChain
from langchain.prompts import PromptTemplate
from langchain.output_parsers import StrOutputParser

# 定义提示模板
prompt = PromptTemplate.from_template(
    "将以下{language}文本翻译成中文：\n{text}"
)

# 构建链
chain = prompt | model | StrOutputParser()

# 执行
result = chain.invoke({
    "language": "英文",
    "text": "Hello, world!"
})
```

## 顺序链

### 简单顺序链

按顺序执行，后一个链以前一个链的输出为输入：

```python
from langchain.chains import SimpleSequentialChain

# 链1：生成书名
chain1 = LLMChain(
    llm=model,
    prompt=PromptTemplate.from_template("为一本关于{topic}的书生成5个书名选项。")
)

# 链2：选择最佳并写简介
chain2 = LLMChain(
    llm=model,
    prompt=PromptTemplate.from_template(
        "从以下书名中选择最佳并写一个简介：\n{book_names}"
    )
)

# 组合
full_chain = SimpleSequentialChain(chains=[chain1, chain2])
result = full_chain.run("人工智能")
```

### 通用顺序链

支持多输入多输出：

```python
from langchain.chains import SequentialChain

chain = SequentialChain(
    chains=[translate_chain, summary_chain, analyze_chain],
    input_variables=["text", "target_language"],
    output_variables=["translated_text", "summary", "analysis"]
)

result = chain.invoke({
    "text": "原始文本",
    "target_language": "中文"
})
```

## 路由链

根据输入动态选择处理路径：

```python
from langchain.chains import LLMRouterChain
from langchain.chains.router.multi_prompt_prompt import MULTI_PROMPT_ROUTER_TEMPLATE

# 定义多个专家链
chains = {
    "math": math_chain,
    "history": history_chain,
    "science": science_chain
}

# 路由链
router_chain = LLMRouterChain.from_llm(
    llm=model,
    router_prompt=MULTI_PROMPT_ROUTER_TEMPLATE
)

# 主链
from langchain.chains import ConversationalChain
full_chain = ConversationalChain(
    memory=memory,
    chains=[router_chain] + list(chains.values())
)
```

## API 链

LLM 调用外部 API：

```python
from langchain.chains import APIChain

# API 文档
api_docs = """
Base URL: https://api.example.com
Endpoints:
- GET /weather?city={city} - 获取城市天气
"""

api_chain = APIChain.from_llm_and_api_docs(
    llm=model,
    api_docs=api_docs,
    verbose=True
)

result = api_chain.run("北京的天气怎么样？")
```

## LangSmith 可观测性

追踪链的执行：

```python
import os
os.environ["LANGCHAIN_TRACING_V2"] = "true"
os.environ["LANGCHAIN_API_KEY"] = "your-api-key"

# 执行链，自动追踪
result = chain.invoke({"input": "..."})

# 查看追踪：https://smith.langchain.com
```

## 小结

链是 LangChain 组织复杂任务的核心抽象。从简单的 LLMChain 到复杂的顺序链和路由链，LangChain 提供了丰富的链类型。掌握链的使用是构建复杂 AI 应用的基础。

# 第四章 AI 应用开发框架

本章介绍主流 AI 应用开发框架的使用方法，包括 LangChain、Semantic Kernel 和 AutoGen。

## 本章内容

- **4.1 LangChain 概述与环境准备**：介绍 LangChain 的核心概念和开发环境搭建
- **4.2 LangChain 模型支持与基本用法**：演示 LLM、Chat 和 Embedding 模型的使用
- **4.3 工具调用基础**：详解如何让大模型调用外部工具
- **4.4 提示模板**：介绍 LangChain 中的提示模板创建和使用
- **4.5 链（Chain）**：详解 LangChain 表达式语言和链的构建
- **4.6 LangChain Agent 与 LangGraph**：介绍 Agent 构建和状态图编程
- **4.7 LangChain RAG 应用开发**：详解 RAG 应用的完整开发流程
- **4.8 Semantic Kernel 概述与环境搭建**：介绍 SK 的核心概念和环境配置
- **4.9 Semantic Kernel 插件与函数**：详解语义函数和原生函数的开发
- **4.10 Semantic Kernel Agent 框架**：介绍 SK 的 Agent 开发方法
- **4.11 Semantic Kernel 流程框架**：详解 SK 的流程编排功能
- **4.12 Semantic Kernel RAG**：介绍 SK 中的 RAG 实现
- **4.13 AutoGen 多智能体框架**：详解 AutoGen 的多智能体协作

## 学习目标

- 掌握 LangChain 的核心组件和开发方法
- 掌握 Semantic Kernel 的插件开发和 Agent 构建
- 掌握 AutoGen 的多智能体协作模式
- 能够使用这些框架开发实际的 AI 应用

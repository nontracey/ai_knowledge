# 4.2 LangChain 模型支持与基本用法

## LangChain 模型支持

### 模型供应商

与 OpenAI、Google、阿里等主流供应商合作，封装了 LLM 类、Chat 类和 Embedding 类模型。

### 接口规范

封装基于 OpenAI API 接口规范，兼容模型可无缝使用。

## LLM 类模型

### 核心功能

接收文本输入，返回文本输出，**无状态**。

### 应用场景

适用于快速问答、文本摘要、信息提取、代码生成等任务。

### 代码示例

```python
from dotenv import load_dotenv
from langchain_community.llms import Tongyi

load_dotenv()

llm = Tongyi(
    model_name="qwen-turbo",
    temperature=0.7
)

response = llm.invoke("你是谁？")
print(response)
```

## Chat 类模型

### 核心功能

接收消息列表，返回 AI 消息，支持多轮对话和角色扮演。

### 应用场景

构建聊天机器人、角色扮演应用、多轮交互任务。

### 代码示例

```python
from langchain_community.llms import Tongyi
from langchain_core.messages import HumanMessage, SystemMessage

chat = Tongyi(model_name="qwen-turbo", temperature=0.7)

messages = [
    SystemMessage(content="你是一个不爱说话的植物学家，回答问题时总是意简言赅。"),
    HumanMessage(content="什么是西瓜？")
]

response = chat.invoke(messages)
print(response)
```

## Embedding 类模型

### 核心功能

将文本映射为高维向量，用于语义搜索和知识库问答，是 RAG 应用的基石。

## 常用参数说明

| 参数 | 说明 |
|------|------|
| temperature | 控制输出的随机性，值越高越有创意 |
| top_p | 动态采样参数 |
| n | 生成的候选数量 |
| max_tokens | 最大生成 token 数 |

## 使用 OpenAI 接口

```python
from langchain_openai import ChatOpenAI
from langchain_core.messages import HumanMessage, SystemMessage

chat = ChatOpenAI(
    base_url="https://api.openai.com/v1",
    api_key="your-api-key",
    model_name="gpt-4",
    temperature=0.7
)

messages = [
    SystemMessage(content="你是一个有帮助的助手。"),
    HumanMessage(content="解释一下什么是机器学习")
]

response = chat.invoke(messages)
print(response.content)
```

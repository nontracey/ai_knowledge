## 模型服务层实现

### 模型提供商服务类

封装不同模型提供商的调用逻辑：

```python
from abc import ABC, abstractmethod
from typing import List, Dict, Any

class BaseModelProvider(ABC):
    @abstractmethod
    def chat(self, messages: List[Dict[str, str]], **kwargs) -> str:
        pass

    @abstractmethod
    def embed(self, texts: List[str]) -> List[List[float]]:
        pass
```

### 模型调用业务方法

实现具体的模型调用逻辑：

```python
class ModelService:
    def __init__(self, provider: BaseModelProvider):
        self.provider = provider

    def chat_with_model(
        self,
        messages: List[Dict[str, str]],
        model: str = "gpt-4",
        temperature: float = 0.7
    ) -> str:
        return self.provider.chat(messages, model=model, temperature=temperature)
```

## MCP 服务生态

### MCP 概述

**MCP（Model Context Protocol）** 是标准化 AI 应用与外部工具交互方式的协议。

### MCP 的价值

- **标准化**：统一的工具调用协议
- **互操作性**：不同 AI 框架间可共享工具
- **可扩展性**：方便添加新工具

### MCP 生态

| 类型 | 说明 | 示例 |
|------|------|------|
| 官方 MCP 服务 | 官方维护的工具 | 文件系统、数据库 |
| 社区 MCP 服务 | 社区贡献 | GitHub、JIRA 集成 |
| 企业自建 MCP | 企业内部工具 | CRM、ERP 系统 |

## 构建 MCP 主机应用

### MCP 客户端集成

```python
from mcp import ClientSession, StdioServerParameters

async def main():
    server_params = StdioServerParameters(
        command="npx",
        args=["-y", "@modelcontextprotocol/server-filesystem", "./data"]
    )

    async with ClientSession(server_params) as session:
        await session.initialize()

        # 调用工具
        result = await session.call_tool("read_file", {"path": "example.txt"})
```

### 文件操作示例

```python
# 读取文件
result = await session.call_tool("read_file", {"path": "document.txt"})

# 写入文件
result = await session.call_tool("write_file", {
    "path": "output.txt",
    "content": "Hello, World!"
})

# 搜索文件
result = await session.call_tool("search_files", {
    "path": "./docs",
    "pattern": "*.md"
})
```


### 重试策略

```python
from tenacity import retry, stop_after_attempt, wait_exponential

@retry(stop=stop_after_attempt(3), wait=wait_exponential(multiplier=1, min=2))
async def call_with_retry(session, tool, params):
    return await session.call_tool(tool, params)
```

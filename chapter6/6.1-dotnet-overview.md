# 6.1 .NET AI 项目概述

## .NET AI 开发概述

.NET 平台提供了丰富的 AI 开发支持，通过 Microsoft.Extensions.AI、Semantic Kernel 等框架，开发者可以轻松地将 AI 能力集成到 .NET 应用中。

## 技术栈

### 核心框架

- **Microsoft.Extensions.AI**：.NET 官方的 AI 抽象层
- **Microsoft.SemanticKernel**：微软推出的轻量级 AI SDK
- **.NET 8+**：最新的 .NET 平台特性

### AI 服务集成

- OpenAI API
- Azure OpenAI Service
- 阿里云百炼（通义千问）
- DeepSeek

### 向量数据库

- Qdrant
- Milvus
- pgvector

## 项目架构

```
├── AI 应用
│   ├── API 层（Minimal API 或 MVC）
│   ├── 业务逻辑层
│   ├── AI 服务层
│   │   ├── Semantic Kernel
│   │   ├── RAG 服务
│   │   └── Agent 服务
│   └── 数据访问层
```

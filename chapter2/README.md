# 第二章 大模型理论基础与应用

本章介绍大语言模型的基础理论、工作流程、训练方法，以及主流大模型的实际应用与部署方式。

## 本章内容

- **2.1 向量化与词向量**：介绍文本向量化的基本概念和词向量的作用
- **2.2 从 RNN 到 Transformer**：梳理循环神经网络到 Transformer 架构的演进历程
- **2.3 大模型工作流程**：详解大模型从输入到输出的完整工作流程
- **2.4 大模型训练：预训练、微调与强化学习**：介绍大模型的三阶段训练方法
- **2.5 大模型的能力边界与局限性**：分析大模型的优势与不足
- **2.6 DeepSeek 深度解析**：深入解析 DeepSeek 系列模型的技术创新
- **2.7 提示词工程**：系统讲解提示词的设计原则与常用框架
- **2.8 国内外主流大模型概览**：介绍 GPT、Claude、Gemini、DeepSeek 等主流模型
- **2.9 大模型云平台与本地部署**：详解云平台计费模式和本地部署方案
- **2.10 用代码调用大模型**：演示 Python 和 .NET 调用大模型的方法
- **2.11 Cursor AI 编程助手**：介绍 Cursor 的使用方法和最佳实践

## 学习目标

- 理解向量化和词向量的基本概念
- 掌握从 RNN 到 Transformer 的技术演进
- 了解大模型的工作流程和训练方法
- 掌握提示词工程的核心技巧
- 熟悉主流大模型的特点和适用场景
- 掌握大模型的云端调用和本地部署方法

# 第二章 大模型理论基础

本章将深入介绍大语言模型的技术原理，包括向量化、神经网络架构、Transformer 工作机制、大模型的训练流程以及提示词工程等核心知识。通过本章的学习，读者将理解大模型为何能够如此"聪明"，以及如何更好地与它交互。

## 本章内容概览

| 小节 | 主题 | 核心知识点 |
|------|------|----------|
| 2.1 | 向量化与词向量 | 文本如何转化为数字表示 |
| 2.2 | 从 RNN 到 Transformer | 深度学习架构的演进历程 |
| 2.3 | 大模型工作流程 | LLM 处理输入生成输出的完整过程 |
| 2.4 | 大模型训练 | 预训练、监督微调、强化学习 |
| 2.5 | 能力边界与局限性 | 幻觉问题、静态知识、安全挑战 |
| 2.6 | DeepSeek 深度解析 | 国产大模型的技术创新 |
| 2.7 | 提示词工程 | 与 LLM 高效交互的技巧 |
| 2.8 | 主流模型对比 | GPT、Claude、Gemini 等模型特点 |
| 2.9 | 云平台与本地部署 | 部署方案的选择与实践 |
| 2.10 | 代码调用大模型 | Python / .NET 集成示例 |
| 2.11 | Cursor AI 编程 | AI 辅助开发实战 |

## 学习目标

- 理解文本向量化的基本原理
- 掌握 Transformer 架构的核心概念
- 了解大模型训练的三阶段方法
- 掌握提示词工程的核心技巧
- 能够使用代码调用大模型 API

# 2.4 大模型训练：预训练、微调与强化学习

大语言模型之所以具备强大的能力，源于其三阶段训练过程。本节将详细介绍预训练、监督微调和强化学习三个阶段。

## 预训练：构建语言知识库

预训练是让模型从零开始"自学语言"的过程，采用无监督学习方法。

### 数据准备

预训练需要海量文本数据，通常包括：

- 网页内容（Wikipedia、新闻、博客等）
- 书籍和文献
- 代码仓库（如 GitHub）
- 对话数据

### 数据预处理

原始数据需要经过多轮处理：

1. **清洗**：去除 HTML 标签、垃圾广告、无意义内容
2. **去重**：删除重复或高度相似的内容
3. **分词**：将文本切分成适合模型的 Token 序列
4. **质量过滤**：保留高质量、有价值的内容
5. **安全审查**：过滤违法和有害内容

### 模型架构

现代大模型几乎都基于 Transformer 架构：

- **位置编码**：注入词语的顺序信息
- **自注意力层**：捕捉词语间的依赖关系
- **前馈网络**：增强表达能力

模型规模从数十亿到数千亿参数不等。

### 自监督训练任务

预训练采用自监督学习，核心任务是预测下一个词：

- **掩码语言建模（MLM）**：类似"完形填空"，将部分词语遮盖，预测被遮盖的词
- **自回归语言建模（ARLM）**：给定前文，预测下一个词

GPT 系列采用自回归方式，从左到右生成文本。

### 优化技术

训练超大模型需要多种优化技术：

| 技术 | 作用 |
|------|------|
| 梯度累积 | 解决显存不足，支持大 batch |
| 混合精度训练 | 加快速度，减少显存 |
| 分布式训练 | 多 GPU 并行，缩短训练时间 |
| 学习率调度 | 先预热后衰减，稳定训练 |
| 检查点保存 | 防止中断导致从头训练 |

## 监督微调：让模型听懂指令

预训练后的模型具备语言能力，但需要进一步训练才能更好地遵循人类指令。

### 什么是监督微调

**监督微调（Supervised Fine-Tuning，SFT）** 是用标注数据对预训练模型进行进一步训练的过程。

### 标注数据的格式

微调数据通常是"指令-响应对"：

```
指令：请将以下英文翻译成中文
输入：Hello, how are you?
输出：你好，你怎么样？
```

### 指令微调技术

为了让模型更好地理解指令，微调时会在数据中加入：

- **指令前缀**：明确告诉模型任务类型
- **输入输出格式**：统一的数据结构
- **多任务混合**：包含多种任务类型

### 多任务学习

微调通常混合多种任务：

- 问答
- 翻译
- 摘要
- 对话
- 代码生成

这让模型具备处理不同任务的能力。

### 常见问题与解决方案

微调过程中可能遇到的问题：

| 问题 | 原因 | 解决方案 |
|------|------|---------|
| 过拟合 | 微调数据过少或训练过度 | 增加数据，多样化训练 |
| 灾难性遗忘 | 新知识覆盖旧知识 | 混合预训练数据 |
| 任务干扰 | 不同任务相互影响 | 任务平衡采样 |
| 风格偏移 | 输出偏离预期格式 | 增加格式约束数据 |

## 强化学习与人类反馈

即使经过微调，模型仍可能生成不符合人类期望的回答。**强化学习从人类反馈中进行学习（Reinforcement Learning from Human Feedback，RLHF）** 能够进一步提升模型与人类偏好的对齐程度。

### 为什么需要 RLHF

监督微调只能学习"正确的回答"，但无法捕捉人类的微妙偏好：

- 什么样的回答更有帮助？
- 什么样的语气更让人舒适？
- 如何在准确性和安全性之间平衡？

### RLHF 的三步骤

#### 第一步：收集人类偏好数据

1. **生成多个回答**：让模型对同一问题生成多个不同回答
2. **人类排序**：人类评估者对回答进行比较和排序
3. **构建偏好数据集**：形成包含排序信息的训练样本

#### 第二步：训练奖励模型

奖励模型（Reward Model）学习预测人类偏好：

- 输入：问题和回答
- 输出：质量评分

训练目标是让高分回答的概率高于低分回答。

#### 第三步：策略优化

使用 **PPO（Proximal Policy Optimization）** 等强化学习算法优化语言模型：

- 让模型生成更可能获得高分回答
- 避免过度优化导致质量下降
- 保持输出的多样性和自然性

### 安全与对齐

RLHF 还被用于确保模型输出的安全性：

- **负向奖励**：对有害内容（暴力、违法等）给予惩罚
- **内容过滤**：检测并阻止有害输出
- **价值观嵌入**：让模型学习社会道德规范
- **对抗训练**：针对刻意绕过安全限制的行为进行训练

## 训练阶段对比

| 阶段 | 目标 | 方法 | 数据类型 |
|------|------|------|---------|
| 预训练 | 学习语言能力 | 自监督学习 | 无标注海量文本 |
| 监督微调 | 任务能力 | 监督学习 | 标注指令数据 |
| RLHF | 人类偏好对齐 | 强化学习 | 人类偏好排序 |

## 小结

大语言模型的训练是一个循序渐进的过程。预训练通过海量文本建立语言知识库，监督微调赋予模型任务执行能力，强化学习让模型的输出更符合人类期望。三个阶段相互配合，共同塑造了今天强大的大语言模型。

# 2.11 Cursor AI 编程助手

## 概述

Cursor 是一个 AI 编程助手，能够帮助开发者更高效地完成编程任务：

- **需求分析阶段**：通过自然语言转化技术方案
- **编码阶段**：智能补全、写代码、创建项目结构
- **测试阶段**：自动生成测试用例、分析代码覆盖率
- **后期维护阶段**：代码重构、性能调优、自动生成技术文档

## 安装与配置

### 下载安装

从官网下载安装 Cursor 并登录账号。

### 基本设置

- Windows 用户在设置里将默认终端 profile 设为 Git bash，避免执行失败
- 在 Cursor Settings → Models 中设置免费模型

## 开发项目流程

### 项目创建

使用 Cursor 创建项目时，需新建文件夹并初始化 Git 仓库，然后通过 AI 对话面板与 Cursor 交互。

### 三种工作模式

- **Agent 模式**：适合复杂任务自动化执行
- **Ask 模式**：适合理解代码
- **Manual 模式**：适合精确编辑

### 使用 Agent 模式

输入详细提示词，Cursor 会生成项目结构和基础代码，可选择接受、拒绝或重新生成修改。

### 提示词示例

```
建一个响应式卡片式新闻网站的基础项目结构，要求如下：
项目结构应包括：
- index.html: 主页面，包含 HTML5 结构和响应式设计 meta 标签
- style.css: 样式文件，包含 CSS Reset、卡片组件样式、主题色切换
- main.js: JavaScript 文件，包含 DOM 操作、动态数据渲染、主题切换
- images/: 图像资源文件夹
- README.md: 项目说明文档

其他要求：
- 使用语义化 HTML 标签
- 页面布局适配 PC 和移动端
- 新闻摘要限制为两到三行，超出部分使用省略号
```

## 代码补全

Cursor 提供智能代码补全功能，支持自动导入设置。

## 进阶项目开发实战

### 销售数据分析系统

读取 CSV（销售数据）→ Pandas 库进行数据清洗、汇总、统计 → 前端负责可视化 → 后端业务逻辑（FastAPI）。

### 开发步骤

1. 编写开发目标、生成架构方案
2. 生成开发文档
3. 使用开发文档指导 Cursor 开发
4. 指导 Cursor 进行阶段性测试

## 沟通技巧

- **明确描述需求**：清晰说明要实现的功能
- **分步骤引导**：将复杂任务拆分为小步骤
- **提供上下文信息**：给出相关文件路径、代码片段
- **迭代优化**：根据生成结果逐步调整
- **有效反馈**：指出问题，让 AI 修正

## 规则系统

使用自己的开发规范，在 Cursor 中配置项目的代码规范和约定，确保生成的代码符合团队标准。

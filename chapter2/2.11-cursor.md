# 2.11 Cursor AI 编程助手

## Cursor 是什么

Cursor 是一款 AI 驱动的代码编辑器，基于 VS Code 构建，集成了先进的大语言模型能力，为开发者提供智能编程辅助。

### 核心价值

Cursor 能够帮助开发者：

- **需求分析阶段**：将自然语言需求转化为技术方案
- **编码阶段**：智能补全、代码生成、项目结构创建
- **测试阶段**：自动生成测试用例、分析代码覆盖率
- **维护阶段**：代码重构、性能调优、文档生成

## 安装与配置

### 下载安装

访问 [cursor.so](https://cursor.so) 下载对应平台的安装包。

### 基本设置

#### Windows 用户

在设置中配置默认终端为 Git Bash，避免命令执行失败：

```
Settings → Terminal → Default Profile → Git Bash
```

#### 模型配置

在 Settings → Models 中配置免费或付费模型：

- 免费模型：适合日常使用
- Pro 模型：更强的代码理解和生成能力

## 三种工作模式

Cursor 提供三种工作模式，适应不同场景：

| 模式 | 适用场景 | 特点 |
|------|---------|------|
| **Agent 模式** | 复杂任务自动化 | 理解高层需求，自动执行多步骤操作 |
| **Ask 模式** | 理解代码逻辑 | 解释代码功能，回答技术问题 |
| **Manual 模式** | 精确编辑 | 精准控制 AI 修改的范围和内容 |

### Agent 模式

适合需要完成多个步骤的复杂任务：

```
输入：创建一个响应式新闻卡片网站

Cursor 会：
1. 创建项目结构
2. 编写 HTML/CSS/JS 代码
3. 实现响应式布局
4. 添加主题切换功能
```

### Ask 模式

适合学习和理解代码：

```
输入：这段代码是做什么的？

Cursor 会：
- 解释代码逻辑
- 指出关键实现细节
- 回答相关技术问题
```

### Manual 模式

适合需要精确控制的编辑：

```
- AI Join Lines
- AI Edit（局部修改）
- AI Inline Chat（内联对话）
```

## 开发流程实践

### 创建新项目

1. 在 Cursor 中新建文件夹
2. 初始化 Git 仓库
3. 使用 AI 对话面板描述需求
4. Cursor 生成初始代码
5. 审查、调整、确认

### 提示词技巧

#### 结构化需求描述

```
差的提示词：建一个网站

好的提示词：
创建一个响应式卡片式新闻网站，要求：
- 技术栈：HTML5 + CSS3 + Vanilla JavaScript
- 页面展示新闻卡片（图片、标题、摘要、链接）
- 卡片式布局，支持 PC 和移动端
- 支持主题切换（默认中国红主题）
- 响应式设计使用 Flexbox 或 Grid
```

#### 分步骤引导

```
第一步：创建项目结构和基础 HTML
第二步：实现 CSS 样式和响应式布局
第三步：添加 JavaScript 交互逻辑
第四步：实现主题切换功能
```

### 代码补全设置

在 Settings → Features 中启用：

- Inline Completion
- Auto Import
- Ghost Text

## 进阶功能

### 规则系统

通过 `.cursorrules` 文件定义项目开发规范：

```
# 代码风格
- 使用中文注释
- 变量命名采用 camelCase
- 函数单一职责

# 技术规范
- 使用 fetch 进行 API 请求
- 错误处理必须完善
- 所有 API 调用需加 try-catch
```

### 数据库集成

Cursor 支持连接数据库：

- 查看数据库结构
- 生成 SQL 查询
- 解释复杂 SQL
- 数据库设计建议

### 调试辅助

- 代码执行预览
- 错误定位和分析
- 修复建议生成

## 最佳实践

### 1. 明确需求描述

提供具体的技术要求和约束条件，避免模糊表述。

### 2. 分步骤进行

复杂任务分解为多个简单步骤，逐步完成。

### 3. 提供上下文

上传相关文件或截图，帮助 AI 更好地理解需求。

### 4. 迭代优化

初次生成不完美是正常的，通过反馈逐步改进。

### 5. 代码审查

AI 生成的代码仍需开发者审查，确保质量和安全性。

## 应用场景

### 快速原型开发

适合 Hackathon 或快速验证想法的场景，Cursor 可以快速生成可运行的项目框架。

### 学习新技术

通过让 AI 解释代码，可以快速理解新框架或新技术的使用方式。

### 代码重构

上传旧代码，让 AI 分析并提出重构建议。

### 技术文档生成

为现有代码生成注释和文档。

## 小结

Cursor 是 AI 编程时代的利器，通过 Agent、Ask、Manual 三种模式，覆盖了从需求分析到代码维护的全流程。合理使用 Cursor 可以显著提升开发效率，但需要注意 AI 生成代码的质量把控。

# 2.10 用代码调用大模型

## Python 调用大模型

### 环境准备

使用 uv 创建虚拟环境并安装依赖：

```bash
uv venv
source .venv/bin/activate  # Linux/Mac
uv pip install openai python-dotenv
```

### OpenAI SDK 方式

```python
from dotenv import load_dotenv
from openai import OpenAI

load_dotenv()

client = OpenAI(
    api_key=os.getenv("API_KEY"),
    base_url=os.getenv("API_BASE")
)

response = client.chat.completions.create(
    model="gpt-4",
    messages=[
        {"role": "system", "content": "你是一个有帮助的助手。"},
        {"role": "user", "content": "你好，介绍一下你自己。"}
    ]
)

print(response.choices[0].message.content)
```

## .NET 调用大模型

### 使用 Microsoft.Extensions.AI

```csharp
using Microsoft.Extensions.AI;

var builder = WebApplication.CreateBuilder(args);
builder.Services.AddChatClient(new OpenAIChatClient(
    new Uri("https://api.openai.com/v1"),
    "gpt-4"));
```

### 使用 Semantic Kernel

```csharp
using Microsoft.SemanticKernel;

var builder = Kernel.CreateBuilder();
builder.AddOpenAIChatCompletion(
    modelId: "gpt-4",
    apiKey: "your-api-key");

var kernel = builder.Build();

var result = await kernel.InvokePromptAsync("解释一下什么是机器学习");
Console.WriteLine(result);
```

### RESTful API 调用

即使没有官方 SDK，也可以通过 HTTP 请求调用：

```csharp
using var client = new HttpClient();
var request = new HttpRequestMessage
{
    Method = HttpMethod.Post,
    RequestUri = new Uri("https://api.openai.com/v1/chat/completions"),
    Headers = { { "Authorization", $"Bearer {apiKey}" } },
    Content = new StringContent(JsonSerializer.Serialize(new
    {
        model = "gpt-4",
        messages = new[] { new { role = "user", content = "Hello!" } }
    }), Encoding.UTF8, "application/json")
};
```

## 流式输出

### Python 流式调用

```python
stream = client.chat.completions.create(
    model="gpt-4",
    messages=[{"role": "user", "content": "写一首诗"}],
    stream=True
)

for chunk in stream:
    if chunk.choices[0].delta.content:
        print(chunk.choices[0].delta.content, end="", flush=True)
```

### .NET 流式调用

```csharp
await foreach (var update in chatClient.GetStreamingResponsesAsync(
    new ChatMessage { Role = ChatRole.User, Content = "讲个笑话" }))
{
    Console.Write(update.Content);
}
```

## 常见问题处理

- **API Key 安全**：使用环境变量存储，不要硬编码
- **错误处理**：添加重试机制处理临时故障
- **超时设置**：合理设置请求超时时间
- **并发控制**：避免过多并发请求导致限流

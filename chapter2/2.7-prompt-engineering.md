# 2.7 提示词工程

## 什么是提示词

提示词（Prompt）是输入给 AI 系统的指令，可以是问题、描述或关键词，目的是引导 AI 按期望的方式思考或创造。

### 为什么提示词如此重要

好的提示词能够：

- 清晰指引 AI 工作方向
- 激发模型的潜在能力
- 显著提升输出质量
- 减少错误和无效迭代

反之，"垃圾进，垃圾出"——质量差的提示词会导致不理想的输出。

## 提示词的基本形式

### 1. 指令式提示词

直接告诉 AI 要做什么，适用于任务目标明确的场景：

```
写一篇关于气候变化的短文。
翻译以下句子为英文：我今天很高兴。
```

### 2. 问答式提示词

通过提问引导 AI 思考并回答：

```
什么是人工智能？
解释量子计算的基本原理。
```

### 3. 填空式提示词

提供句子框架让 AI 填充：

```
我最喜欢的动物是____，因为____。
```

### 4. 对话式提示词

模拟人与人之间的自然对话：

```
你好！能跟我聊聊今天的天气吗？
我们来讨论一下最近的电影。
```

### 5. 创意生成式提示词

给予 AI 更大发挥空间：

```
写一首描绘未来世界的诗。
设计一个创新的手机应用概念。
```

### 6. 思维链提示词

引导 AI 逐步推理：

```
让我们一步步来思考这个问题。首先...
```

## 编写高质量提示词的关键

### 1. 明确目标

在写提示词之前，先明确：

- 你希望 AI 完成什么任务？
- 期望的输出是什么类型？
- 有哪些具体要求或限制？

目标越清晰，AI 越容易理解并给出精准回应。

### 2. 提供充分的背景信息

背景信息帮助 AI 理解语境：

- 受众特点：面向专业人群还是普通大众？
- 领域背景：涉及哪些行业或专业知识？
- 上下文关系：与什么相关联？

### 3. 明确输出要求

清楚告诉 AI 你期望的输出：

- **内容类型**：文章、代码、表格、列表？
- **格式要求**：字数限制、结构框架？
- **风格语气**：正式/随意、专业/通俗？
- **侧重点**：强调什么、避免什么？

## 实用技巧

### 指令清晰化

**使用明确动词开头：**

```
差：看看这段代码有没有问题
好：审查这段代码，重点检查性能优化和安全漏洞
```

**使用结构化表达：**

```
请按以下格式输出：
1. 问题分析
2. 解决方案
3. 代码示例
```

### 角色扮演与人设

给 AI 设定角色，帮助它更好地理解任务语境：

```
你是一位有20年教学经验的高中数学老师，擅长将复杂概念简单化。
你现在是一名获得过多个文学奖项的科幻小说作家，擅长世界观构建。
你是一位专注用户体验的产品经理，参与过多个独角兽公司的产品设计。
```

### 少样本学习

在提示词中提供示例，帮助 AI 理解期望的格式或风格：

```
请按照以下格式撰写产品描述：

输入：iPhone 14 Pro
输出：突破性的创新设计，卓越的性能表现。搭载A16仿生芯片，4800万像素主摄像头提供专业级拍摄体验。

输入：MacBook Air
输出：轻薄优雅的便携之选，出众的续航能力。M2芯片带来强劲性能。

现在请为以下产品撰写描述：
输入：iPad Pro
```

### 约束性提示

通过具体限制控制输出：

```
请写一篇产品推广文案，要求：
- 字数限制在100字以内
- 必须包含关键词：创新、智能、便携
- 使用专业但不晦涩的语言
- 结构为：痛点-解决方案-产品优势
- 以号召性用语结尾
```

## 思维链提示

对于复杂问题，引导 AI 逐步推理：

```
请分析这个问题，并展示你的思考过程：
1. 首先，让我们理解问题的关键信息...
2. 接下来，我们可以采用以下方法解决...
3. 最后，让我们验证答案的合理性...
```

## 迭代优化

提示词工程是一个反复调整的过程：

1. **写初版**：先写出基本的提示词
2. **测试**：观察 AI 的输出
3. **分析**：找出与期望的差距
4. **调整**：改进提示词的表述或结构
5. **重复**：直到达到满意效果

## 推理模型的提示词特点

对于 DeepSeek-R1 等推理模型，提示词策略有所不同：

### 简洁直接

推理模型擅长自主推理，提示词应更简洁：

```
差：如果可以的话，我想请你帮忙看看这段代码是否存在什么问题，如果方便的话可以给出一些改进建议
好：分析这段代码的问题并提供改进建议
```

### 信任模型能力

不要过度规定步骤，让模型自己推理：

```
差：首先分析用户需求，然后进行市场调研，接着制定营销策略，最后预估投资回报率。请按照这些步骤帮我评估这个商业计划
好：评估这个商业计划的可行性
```

### 明确目标而非步骤

```
差：帮我看看这份市场报告
好：分析2023年Q4电商行业报告，重点关注用户增长趋势和竞品策略，给出未来6个月的市场机会
```

## 小结

提示词工程是与大语言模型高效交互的关键技能。通过明确目标、提供背景、规定输出格式，并运用角色扮演、少样本学习、思维链等技巧，可以显著提升 AI 输出质量。提示词工程是一个迭代优化的过程，需要不断尝试和调整。
